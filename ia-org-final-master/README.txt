[ENG]

ARGS if there is 1 AI: game.py [PLAYER1] [PLAYER2] [ALGORITHM] [DEPTH] [AI-THINKING]

ARGS if there is 2 AI: game.py [PLAYER1] [ALGORITHM] [DEPTH] [PLAYER2] [ALGORITHM] [DEPTH] [AI-THINKING]

PLAYER 1 AND PLAYER 2: h - human or b - bot
ALGORITHM: minimax - minimax algorithm or negamax - negamax algorithm
DEPTH: int - number of movements that are used to forecast
AI-THINKING: t - true, you can see AI thought process or f - false, you can not see AI thought process

Basic example:

H vs B python game.py h b minimax 2 f
B vs B python game.py b negamax 3 b minimax 2 f

[ESP]

ARGS si existe 1 IA: game.py [JUGADOR1] [JUGADOR2] [ALGORITMO] [PROFUNDIDAD] [PROCESO-IA]

ARGS si existe 2 IA: game.py [JUGADOR1] [ALGORITMO] [PROFUNDIDAD] [JUGADOR2] [ALGORITMO] [PROFUNDIDAD] [PROCESO-IA]

JUGADOR1 y JUGADOR2: h - humano o b - bot
ALGORITMO: minimax - algoritmo minimax or negamax - algoritmo negamax
PROFUNDIDAD: int - número de movimientos utilizados para la previsión
PROCESO-IA: t - true, puedes ver el proceso de escoger movimiento o f - false, no puedes ver el proceso de escoger movimiento

Ejemplo básico:

H vs B python game.py h b minimax 2 f
B vs B python game.py b negamax 3 b minimax 2 f
